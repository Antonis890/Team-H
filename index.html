Project Scope

This project involves the design and development of a Treasure Hunt mobile web application that enables users to participate in a location-based quiz game. Clues are dynamically retrieved from a provided Treasure Hunt API, and the application is built using HTML5, CSS, and JavaScript.

Project Goals

  • Develop a mobile web application that enables users to complete a full treasure hunt session using the provided API
  • Implement location-based functionality by requiring users to provide geographic data with each response
  • Ensure good usability and teamwork by applying UI/UX principles, testing the application, and collaborating effectively using GitHub
